== Installation ==

How to install BuscaPé WP Related Products

e.g.

1. Upload `buscape-wp-related-products` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the plugin through the 'BuscaPé WP Related Products' menu in WordPress
